# Project plan — CLL TDA ML research

## Scientific objective

Build a dynamic mapping linking biological trajectories to clinical trajectories in chronic lymphocytic leukemia, in order to provide a geometric interpretation of CLL heterogeneity.

## Main direction

Start with the biology-to-clinical direction:

1. biological and clinical variables;
2. classical Machine Learning baselines;
3. geometric patient space;
4. topological feature extraction;
5. hybrid TDA + ML models;
6. clinical and biological interpretation.

## Core comparison

Compare:

1. classical ML model;
2. TDA-only model;
3. hybrid classical features + TDA features model.

## Recommended initial endpoints

Priority endpoints:

1. time-to-first-treatment;
2. risk of progression;
3. progression-free survival if survival data are available.

## Interpretation targets

Priority clinical and biological interpretation axes:

1. IGHV status;
2. TP53 mutation;
3. del(17p);
4. Rai/Binet stage;
5. beta-2 microglobulin;
6. lymphocyte count;
7. treatment or progression status.
